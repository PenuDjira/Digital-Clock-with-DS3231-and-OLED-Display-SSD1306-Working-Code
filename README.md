# SYSTEM OVERVIEW -- WHAT WE BUILD???
Digital Clock with DS3231 RTC Module and SSD1306 OLED Displays

# HOW IT WORKS??
DS3231 RTC Module read current time then send to the MCU ATMega 328p in Arduino Board and send to the OLED display.
Both of the module communicate with MCU using I2C

# HOW TO RUN THIS PROJECT?
 1. Download of clone this repo.
 2. Wiring the system follow the schematic in main directori.
 3. Download the Library that project need (Adafruit_SSD1306 and RTClib)
 4. Make sure the OLED address match with the resolution you use.
 5. Upload your code.
